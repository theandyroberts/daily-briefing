# Andy's Market Brief

A daily market briefing dashboard published on GitHub Pages. Tracks EV/AV stocks, macro signals, and trade setups — generated by **Rusty**, an AI-powered market brief engine.

## Live Site

Visit the GitHub Pages deployment to see the latest brief.

## How It Works

1. `generate_brief.py` fetches live prices from Yahoo Finance and generates AI-powered analysis via the Grok API (xAI)
2. Output is written to `brief.json`
3. `index.html` reads `brief.json` and renders the briefing dashboard
4. GitHub Pages serves the static site directly from the `main` branch

## Running the Generator

```bash
pip install -r requirements.txt
export XAI_API_KEY="your-xai-api-key"
python generate_brief.py
```

The script will:
- Fetch current prices for the watchlist (TSLA, LMND, RIVN, PONY, MBLY, SPY, QQQ, USO)
- Call Grok to generate macro analysis, sector notes, and trade setups
- Write everything to `brief.json`

## Automation

The brief is generated automatically via openclaw cron, running weekdays at 7:30 AM ET. The cron job runs `generate_brief.py`, commits the updated `brief.json`, and pushes to `main` — GitHub Pages picks up the change automatically.
